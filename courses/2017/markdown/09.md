### Soal 1

```java

class App {
  public static void main(String args[]) {
    String food = "pecel";
    int x = 2, y = -2;
    if (x < 0 || y > 0) {
    	food = "bakso";
    }
    System.out.println("I love " + food);
  }
}

```
