# Nested Loop

<!-- .slide: data-background="images/11/success.gif" -->

---

```java
class App {
  public static void main(String args[]) {
    int N = 50;
    for (int i = 0; i < N; i++) {
      for (int j = 0; j < N; j++) {
        System.out.print("*");
      }
      System.out.println();
    }
  }
}
```
